# Financial Platform Hebrew

פרויקט React + Vite לניהול כספים בעברית עם RTL מלא.

## מה יש במערכת
- דשבורד הכנסות/הוצאות
- הוספה, עריכה ומחיקה של עסקאות
- עסקאות קבועות / משתנות
- יצירת עסקאות קבועות לחודש הנוכחי
- שמירה אוטומטית ב-localStorage
- דוחות וגרפים
- ייצוא ל-CSV
- הדפסה / שמירה כ-PDF

## הרצה מקומית
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## העלאה ל-GitHub
1. צור Repository חדש ב-GitHub.
2. חלץ את ה-ZIP.
3. פתח טרמינל בתוך התיקייה והריץ:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin YOUR_REPOSITORY_URL
git push -u origin main
```
